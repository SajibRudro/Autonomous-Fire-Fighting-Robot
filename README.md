# 🚒 Autonomous Fire Fighting Robot

An autonomous fire-fighting robot built using Arduino UNO and ESP32.

## 🔥 Features
- Fire detection using 3 IR flame sensors
- Autonomous movement toward fire
- Servo-controlled water pump
- WiFi manual control using ESP32
- Auto/Manual mode switching

## 🛠 Technologies Used
- Arduino UNO
- ESP32 DevKit V1
- L298N Motor Driver
- IR Flame Sensors
- Servo Motor
- Relay Module
- C++
